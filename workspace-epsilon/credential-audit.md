## 五、安全发现

### 🔴 高风险

1. **.env 文件未被 .gitignore 保护**
   - 风险：误执行 `git add .` 可能导致密钥泄露到代码仓库
   - 影响：所有 API 密钥暴露
   - 建议：立即添加 `.env` 到 `.gitignore`

### 🟡 中风险

2. **两个 .env 文件分散存储**
   - 问题：`~/.openclaw/.env` + `~/.openclaw/workspace/.env`
   - 风险：管理混乱，可能遗漏
   - 建议：合并到 `~/.openclaw/.env`（单一来源原则）

3. **环境变量对同用户进程可见**
   - 问题：同一用户下所有进程可读取环境变量
   - 风险：恶意脚本可窃取密钥
   - 建议：使用 secrets manager（长期）或限制进程权限（短期）

### 🟢 低风险

4. **文件权限配置正确** ✅
5. **日志无密钥泄露** ✅
6. **credentials 目录权限良好** ✅
