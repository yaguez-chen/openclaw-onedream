## 七、探索方法

1. 从 ClawHub 搜索相关技能
2. 使用 skill-guard 进行安全检查
3. 阅读文档和用户评价
4. 在测试环境中试用
5. 评估是否适合小龙虾世界

### 2. 测试环境
- 在 `custom/active/` 目录中测试
- 使用 skill-test 技能进行功能测试
- 记录测试结果和发现的问题

- 在 `reviews/` 目录中记录评估结果
- 分享经验到 shared-knowledge
