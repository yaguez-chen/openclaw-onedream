# 自定义技能开发目录

## 目录结构

```
custom/
├── README.md           # 本文件
├── templates/          # 技能模板
│   └── basic-skill.md  # 基础技能模板
└── active/             # 正在开发的技能
```

## 使用说明

### 开发新技能

1. **复制模板**
   ```bash
   cp templates/basic-skill.md active/my-new-skill.md
   ```

2. **编辑技能文档**
   - 修改基本信息
   - 编写功能描述
   - 添加使用示例
   - 记录配置选项

3. **测试技能**
   - 使用 skill-test 技能进行测试
   - 记录测试结果
   - 修复发现的问题

4. **发布技能**
   - 完善文档
   - 安全检查（使用 skill-guard）
   - 发布到 ClawHub 或本地部署

### 技能开发流程

1. **需求分析** → 确定技能要解决的问题
2. **模板创建** → 基于模板创建初始文档
3. **功能开发** → 实现技能功能
4. **测试验证** → 全面测试技能
5. **文档完善** → 编写完整的使用文档
6. **安全检查** → 进行安全评估
7. **发布部署** → 发布到目标平台

### 最佳实践

- 保持技能文档简洁明了
- 提供清晰的示例代码
- 记录所有配置选项
- 包含故障排除指南
- 定期更新和维护
