## 二、信息分类

### 🔴 紧急信息（红色通道）
- **定义**：需要立即处理，延迟会造成损失或安全风险
- **示例**：系统异常、安全事件、梦想家紧急需求、重大决策需立即批复
- **处理**：直接通知相关人 + 西塔 + 梦想家，同步记录到世界状态看板
- **响应时间**：15分钟内

### 🟡 重要信息（黄色通道）
- **定义**：需要关注但不需要立即行动
- **示例**：任务进度更新、里程碑达成、发现重要问题、资源需求
- **处理**：更新到状态看板 + 每日简报汇总
- **响应时间**：当日内

### 🟢 日常信息（绿色通道）
- **定义**：常规工作进展和日常事务
- **示例**：日常任务完成、文档更新、小型优化、学习笔记
- **处理**：记录到状态看板，每日简报汇总
- **响应时间**：24小时内

### 🔵 通知信息（蓝色通道）
- **定义**：全员通知，不需回复
- **示例**：新政策、流程变更、世界规则更新、新公民加入
- **处理**：通过西塔统一广播
- **响应时间**：看到即可

## 三、信息流转机制

### 3.1 自上而下（梦想家 → 全体）

```
梦想家 → 西塔 → 分类分发 → 各公民
         ↓
    （西塔判断信息类型，决定传播范围）
```

**规则**：
- 梦想家直接下达的指令，西塔15分钟内分发
- 需要多公民协作的任务，西塔协调资源
- 明确执行者，设定截止时间

### 3.2 自下而上（公民 → 梦想家）

```
公民 → 西塔（汇总） → 梦想家
  ↓
（常规情况走汇总路线）
  ↓
（紧急情况可直接联系梦想家）
```

**规则**：
- 日常事务：先报西塔，由西塔汇总后统一向梦想家汇报
- 紧急事务：可以直接联系梦想家，同时告知西塔
- 需要梦想家决策的：整理好信息后通过西塔提交

### 3.3 横向协作（公民 ↔ 公民）

```
公民A → sessions_send → 公民B
        （同时通知西塔备案）
```

**规则**：
- 伙伴间直接通过 `sessions_send` 通信
- 协作完成后将结果告知西塔备案
- 涉及多个公民的项目，由约塔统一协调

### 3.4 信息存储与归档

```
实时状态 → /world-status-board.md（看板）
每日汇总 → /shared-knowledge/operations/briefings/YYYY-MM-DD.md（简报）
重大事件 → /shared-knowledge/operations/events/（事件记录）
决策记录 → /shared-knowledge/decisions/（决策日志）
```

## 四、紧急信息处理流程

### 4.1 触发条件
- 系统故障或异常行为
- 安全事件或潜在风险
- 梦想家的紧急需求
- 阻塞性问题影响多公民

### 4.2 处理流程

```
1. 发现紧急情况
   ↓
2. 判断紧急级别（红色/黄色）
   ↓
3. 立即通知：
   - 西塔（必须）
   - 梦想家（红色必须，黄色可选）
   - 相关公民（根据情况）
   ↓
4. 记录到世界状态看板（标记🚨）
   ↓
5. 西塔协调处理资源
   ↓
6. 处理完成后更新状态 + 归档
```

### 4.3 谁可以发起紧急通知
- **任何公民**都可以发起紧急通知
- 不需要先获得批准
- 发起后需补充详细信息

## 五、日常信息同步机制

### 5.1 每日简报（见 daily-briefing-process.md）
- **时间**：每天上午 9:00（北京时间）
- **收集**：每日 20:00-24:00 各公民提交当日报告
- **汇总**：西塔负责汇总整理
- **分发**：通过飞书推送简报给梦想家 + 全体公民

### 5.2 周度总结
- **时间**：每周日晚上
- **内容**：本周完成事项、下周计划、关键问题
- **负责人**：约塔（项目管理官）

### 5.3 事件即时更新
- 任何公民完成重要任务 → 更新世界状态看板
- 任何公民发现重要问题 → 通知西塔
- 西塔判断是否需要广播

## 六、决策信息传达

### 6.1 需要梦想家决策的情况
- 重大方向性选择
- 资源分配决定
- 冲突解决
- 新政策制定
- 预算/权限变更

### 6.2 决策提交格式

```
📋 决策请求

标题：[简短描述]
背景：[为什么需要决策]
选项：
  A. [选项1] — [优缺点]
  B. [选项2] — [优缺点]
建议：[推荐选项及理由]
截止：[需要回复的时间]
提交人：[公民名]
```

### 6.3 决策传达流程

```
提交人 → 西塔（格式化） → 梦想家（决策）
                           ↓
                    西塔（传达决定）→ 相关公民
                           ↓
                    艾塔（记录到决策日志）
```

## 八、沟通原则

1. **透明优先**：信息默认公开，除非涉及隐私
2. **及时响应**：不拖延，不确定就问
3. **简洁清晰**：说重点，附带必要上下文
4. **留痕可溯**：重要沟通要有记录
5. **各司其职**：分工明确，减少重复沟通
