## 🎯 贡献目标

让每个伙伴都能轻松贡献知识，让知识仓库成为集体智慧的宝库。

## 🚀 如何贡献知识

### 第一步：准备贡献
1. **确定知识类型** - 选择正确的目录
2. **检查重复** - 搜索是否已有类似知识
3. **收集信息** - 确保信息准确完整

### 第二步：编写知识文档
1. **选择模板** - 使用对应的模板（见下文）
2. **填写内容** - 按照格式要求编写
3. **添加标签** - 使用合适的标签便于搜索

### 第三步：提交贡献
1. **保存文件** - 放在正确的目录
2. **更新索引** - 在 `index.md` 中添加条目
3. **通知管理员** - 告知艾塔新贡献

## 📊 知识质量标准

### ✅ 必须满足
1. **准确性** - 信息经过验证，来源明确
2. **完整性** - 包含必要背景和详细步骤
3. **可读性** - 结构清晰，语言简洁
4. **时效性** - 标注日期，定期更新

### 📈 鼓励添加
1. **示例代码** - 实际可运行的代码
2. **截图/图表** - 可视化说明
3. **相关链接** - 参考资料
4. **测试用例** - 验证方法

### ❌ 避免
1. **猜测信息** - 未经验证的内容
2. **重复内容** - 已有类似知识
3. **过时信息** - 已失效的内容
4. **机密信息** - 不应公开的内容

## 🔄 知识审核流程

### 审核角色
- **知识管理员（艾塔）**：最终审核和批准
- **架构专家（拉姆达）**：架构知识审核
- **其他伙伴**：同行评审

### 审核步骤
1. **提交贡献** - 伙伴提交新知识
2. **初步检查** - 艾塔检查格式和完整性
3. **技术审核** - 相关专家审核技术内容
4. **反馈修改** - 根据反馈修改完善
5. **最终批准** - 艾塔批准并更新索引
6. **通知相关方** - 通知可能受益的伙伴

### 审核时间
- **简单贡献**：24小时内审核
- **复杂贡献**：48小时内审核
- **紧急知识**：立即审核
