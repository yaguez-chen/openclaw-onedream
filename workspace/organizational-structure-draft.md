## 二、汇报关系

```
梦想家（董事长）

    ├── 贝塔（监察安全部）→ 监控/上报
    ├── 西塔（沟通部）→ 简报/协调
    ├── 约塔（项目管理部）→ 流程/追踪
    ├── 艾塔（知识管理部）→ 归档/索引
    ├── 艾普西隆（安全部）→ 审计/响应
    └── 业务组（伽马/拉姆达/卡帕/泽塔）
```

## 三、决策流程

**常规决策：** 方向负责人提出 → 阿尔法审批 → 执行 → 监察 → 汇报

**重大决策：** 发现问题 → 阿尔法设计方案 → 请示梦想家 → 梦想家决策 → 阿尔法执行 → 汇报结果

**紧急决策：** 风险发现 → 安全部/监察部立即上报 → 阿尔法评估 → 直接通知梦想家 → 先处置后补流程

## 四、核心制度（待约塔细化）

1. **交付物制度** — 所有产出必须写文件，口头汇报不等于交付
2. **日报制度** — 每人每日写日志，西塔汇总发梦想家
3. **任务确认制度** — 完成需写结果文件，贝塔每日检查
4. **协作规范** — 负责人统筹，协作人配合，跨方向约塔协调
