## qmd 技能安装 (2026-03-16 13:49-14:44)
- **技能名：** qmd（Quick Markdown Search）
- **作者：** steipete（@tobilu）
- **功能：** 本地 Markdown 搜索/索引 CLI，BM25 + 向量语义搜索 + LLM 重排序
- **特色：** MCP 模式、GPU 加速（Vulkan）、完全本地运行
- **安装过程：**
  1. clawhub install qmd ✅
  2. bun 安装 ✅（从 bun.sh 下载，约 100MB）
  3. bun install -g qmd ✅（从 GitHub 安装 @tobilu/qmd）
  4. bun run build ✅（TypeScript 编译，依赖安装后构建成功）
- **遇到的问题：**
  - bun 全局安装后 dist 文件缺失，需要手动 build
  - Git clone 走清华镜像源导致 502（系统级 redirect，无法绕过）
  - 解决方案：在已安装包目录下 `bun install && bun run build`
- **状态：** ✅ 已就绪，`qmd status` 正常
- **GPU：** Intel UHD 620 (Vulkan)，1.1GB 空闲/5.7GB 总计
