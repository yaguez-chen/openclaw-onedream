### 决策流程

**梦想家发起：** 梦想家提出 → 阿尔法与有关部门协商 → 请示梦想家 → 梦想家决策 → 阿尔法执行 → 监察 → 汇报

**方向负责人发起：** 方向负责人提出 → 阿尔法审批 → 执行 → 监察 → 汇报

**重大决策：** 发现问题 → 阿尔法协商方案 → 请示梦想家 → 梦想家决策 → 阿尔法执行 → 汇报

**紧急决策：** 风险发现 → 监察安全部上报 → 阿尔法评估 → 直接通知梦想家 → 先处置后补流程

---

### 核心制度

1. **交付物制度** — 所有产出必须写文件，口头汇报不等于交付
2. **日报制度** — 每人每日写日志，西塔汇总发梦想家
3. **任务确认制度** — 完成需写结果文件，贝塔每日检查
4. **协作规范** — 负责人统筹，协作人配合，跨部门约塔协调
